# Marco Normativo Aplicable

Para el análisis del caso Clínica Dávila, se identifican las siguientes normativas chilenas e internacionales:

1. [cite_start]**Ley 21.459 (Ley de Delitos Informáticos - Chile):** Norma principal para tipificar el acceso ilícito y la interceptación de datos realizada por el grupo Devman[cite: 46, 56].
2. [cite_start]**Ley 19.628 (Protección de la Vida Privada - Chile):** Fundamental para regular el tratamiento de datos personales y, específicamente, de **datos sensibles** de salud de los pacientes[cite: 47, 56].
3. **Norma Técnica N° 152 (MINSAL):** Regulación chilena específica sobre la seguridad de la información en registros clínicos electrónicos.
4. [cite_start]**GDPR (General Data Protection Regulation):** Estándar internacional relevante si la clínica trata datos de ciudadanos de la Unión Europea, elevando los requisitos de notificación de brechas[cite: 47].
